## File Naming Convention

```

```

**Naming Rules:**

* Base category name in Spanish (product type)
* Underscore-separated variant descriptors for different preparations
* Numeric suffix (e.g., `2`) for generic variations
* All lowercase with Spanish characters preserved
* Consistent `.jpg` format

**Examples:**

* `alitas_mostaza_miel.jpg` = wings + honey mustard variant
* `hamburguesa_doble_queso.jpg` = hamburger + double cheese variant
* `hotdog_clasico.jpg` = hot dog + classic variant

### XMP Metadata Package

XML-based metadata following Adobe XMP specification. Structure:

```

```

**Key Fields:**

* `photoshop:Credit` - Attribution to Getty Images/iStockphoto
* `GettyImagesGIFT:AssetID` - Unique identifier for tracking
* `xmpRights:WebStatement` - License agreement URL
* `plus:DataMining` - Data mining restrictions
* `dc:creator` - Photographer/creator name
* `dc:description` - Image description
* `plus:Licensor` - Licensor URL

## Licensing and Rights Management

All images are governed by Getty Images/iStockphoto licensing agreements.

### License Tracking

```

```

### Key Licensing Components

**Asset Tracking:**

* Each image has unique `GettyImagesGIFT:AssetID`
* IDs traceable to original iStockphoto listings
* Enables verification of legitimate usage

**Creator Attribution:**

* `dc:creator` field identifies photographer
* Examples: emrah_oztas, semenovp, Zufar Kamilov, Cemal Onur Ucan, monticelllo

**License Agreement:**

* `xmpRights:WebStatement` links to license terms
* URL: `https://www.istockphoto.com/legal/license-agreement`
* Includes UTM tracking parameters for verification

**Data Mining Restrictions:**

* `plus:DataMining`: `DMI-PROHIBITED-EXCEPTSEARCHENGINEINDEXING`
* Prohibits automated content scraping
* Exception: Search engine indexing allowed

### Licensor URLs

Each image includes a direct licensor URL for verification:

**Pattern:** `https://www.istockphoto.com/photo/license-gm{AssetID}-/?utm_medium=organic&utm_source=google&utm_campaign=iptcurl`

**Examples:**

* Cheesecake: `license-gm1146488934-`
* Wings with sauce: `license-gm1285462667-`
* Honey mustard wings: `license-gm1098318622-`
* Brownie: `license-gm881733936-`
* Beer: `license-gm1337471276-`
